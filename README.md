# Python Practice Projects 🐍
A collection of Python projects from my learning journey..

## 📂 Projects
### 1. `discount_calculator.py` — Instant Discount System
Calculates a discount based on the purchase amount:
- Less than 100 → 0% discount
- 100–499 → 10% discount
- 500 or more → 20% discount
  
### 2. `job_eligibility_checker.py` — Job Eligibility Portal
Filters applicants for a "Junior Python Developer" position based on:
- Python proficiency (required)
- Years of experience / number of projects
- University degree in Computer Science or completion of a bootcamp
Applicant is accepted if they master Python **and** (have 2+ years of experience/projects **or** hold a degree/bootcamp certificate).

### 3. `atm_simulator.py` — Smart ATM Simulator
Simulates a basic ATM flow:
1. PIN verification
2. Choice between withdrawing cash or checking balance
3. Withdrawal validation (insufficient balance check)

## 🚀 How to Run
Make sure you have Python 3 installed, then run any project with:
```bash
python discount_calculator.py
python job_eligibility_checker.py
python atm_simulator.py
```
