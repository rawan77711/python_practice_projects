# 🏦 Python Bank Account Management System

## 📌 About
A simple **Bank Account Management System** built using only Python basics:
- Variables
- Input / Output
- Conditions (`if / elif / else`)
- Loops (`while`)
- Functions
- Dictionaries

🚫 No OOP, No Database, No Files, No external libraries.
---

## ⚙️ Features
- Register new account (username + password)
- Login with 3 attempts limit
- Bank Menu:
  - Check Balance
  - Deposit (with receipt)
  - Withdraw (with receipt)
  - Transfer (with receipt)
  - Change Password
  - Logout
- Input validation (no empty fields, password ≥ 6 chars, sufficient balance, etc.)
- Cancel operations by entering `0`
---

## ▶️ How to Run
   ```bash
   python3 main.py
   ```
